Description

Medigment is a web application designed to streamline the medication management process for patients and healthcare providers. The platform allows users to easily track their medications, set reminders, and access important information about their prescriptions. Built with a focus on user experience, Medigment utilizes modern web technologies to provide a responsive and intuitive interface.

Technologies Used

  • PHP
  • HTML
  • CSS
  • Bootstrap
  • JavaScript
  • jQuery
  • MySQL
